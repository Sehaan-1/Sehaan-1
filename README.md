## Engineering Principles (how I build)
- **Evaluation before iteration**: baseline → controlled changes → measured uplift
- **Reproducible runs**: deterministic configs, clean environments, documented setup
- **Production posture**: structured logging, failure handling, and clear boundaries
- **Readable systems**: tight README, diagrams where useful, and minimal “magic”
