### 🔍 [TenderPulse](https://github.com/yourusername/tenderpulse)
> **Production-grade eProcure Tender Intelligence Engine**

Automated extraction of **50,000+ public tenders** from India's government procurement portal (eProcure/CPPP). Built to survive the most hostile scraping environment in Indian e-government infra.

- ⚙️ **Playwright** headless browser with ASP.NET session management
- 💾 **SQLite task queue** with heartbeat, stale detection & crash recovery
- 🔁 **9-bucket retry taxonomy** — each error class has its own backoff schedule
- 🧮 **Indian money/date normalization** — ₹/lakh/crore → `Decimal`, Indian formats → `datetime`
- 📤 **Dual export** — CSV (analysts) + JSONL (BigQuery / streaming pipelines)
- ✅ `mypy --strict` + Ruff enforced · 15 test modules · SIGTERM graceful shutdown
